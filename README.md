# My Blog Backend

Короткий бэкенд для блога, реализованный на Spring Boot. Приложение предоставляет REST API для работы с постами, комментариями и лайками, а также автоматическую документацию Swagger/OpenAPI.

## Основные возможности
- управление постами: создание, обновление, удаление, загрузка и получение изображений;
- управление комментариями к постам;
- инкремент лайков у поста;
- health-check эндпоинт `/api/health`.

## Как запустить локально
1. Убедитесь, что установлены Java 21 и Maven.
2. Выполните `mvn spring-boot:run`.
3. Swagger UI будет доступен по адресу `http://localhost:8080/swagger-ui/index.html`.

## Стек
- Java 21 / Spring Boot 3;
- Spring Data JPA + H2;
- Springdoc OpenAPI.
